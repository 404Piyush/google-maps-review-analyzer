🛠️ Three ways to run it

    

        

            
①

            
Scrape

            
Puppeteer + stealth + rotating proxies. Falls back across proxies in parallel.

            
node index.js --parallel-proxies=2 --fast

        

        

            
②

            
Places API

            
Official Google Places API. No scraping, no proxies. $200/mo free credit.

            
GOOGLE_PLACES_API_KEY=... node places-api.js --text-search="Joe's Pizza"

        

        

            
③

            
Analyze

            
Batched Ollama or OpenRouter free models. Auto-detects GPU vs CPU.

            
node topic-analysis.js --model=fast --batch-size=10
