01

What it does.

Three jobs. One tool. Pick your flavor.

①

Scrape the open web

Puppeteer with stealth and a rotating proxy race. Falls back to the next proxy the moment one stalls — no more waiting 90 seconds on a dead endpoint.

$ node index.js --parallel-proxies=2 --fast
Puppeteer Stealth Proxy race
②

Or use the official API

Same output schema, no scraping drama. $200/mo free credit covers most projects.

$ node places-api.js
0 faster than v1.1
03

Analyze

④

Local or cloud LLM

Ollama on your laptop, or any OpenRouter model in the browser tab. Auto-detects GPU vs CPU; streams output as it arrives.

gemma2:2b qwen3:8b Nemotron 550B Llama 3.3 70B
⑤

One CLI

Scrape, fetch, analyze, demo — all under one binary. Aliases for everything.

$ npx gmaps-analyzer scrape:api
$ npx gmaps-analyzer analyze --model=fast

        03
        
The pipeline.

        
Three stages. Cached at every step.

    


    
    
        
  1. 
            
    ①
    
            
    
                
    Scrape or fetch
    
                
    Puppeteer stealth + parallel proxies, or the official Places API. URL cache skips repeats for 24 hours.
    
                
    $ node index.js --parallel-proxies=2 --fast
# or
$ node places-api.js --text-search="Joe's Pizza"
    
            
    
        
    2. 
        
  2. 
            
    ②
    
            
    
                
    Analyze
    
                
    Batched LLM calls (10 reviews / batch). GPU auto-detected on Apple Silicon + CUDA. Resumes from intermediate file on crash.
    
                
    $ node topic-analysis.js --model=fast --batch-size=10
    
            
    
        
    3. 
        
  3. 
            
    ③
    
            
    
                
    Report
    
                
    Markdown executive report with sentiment breakdown, top topics, and a written summary. Streams to your terminal or the demo tab.
    
                
    output/executive-report.md
output/extracted-reviews.json
output/intermediate-analysis.json
